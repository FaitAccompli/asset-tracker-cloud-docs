Getting startedÂ¶

Before you start the setup of the Asset Tracker Cloud Example on AWS, ensure that:

  • your system fullfills the necessary system requirements

  • you have installed a direnv plugin for your shell

  • you have an AWS Account and AdministratorAccess, if not here is how you create and activate a new AWS account

Then complete the following steps:

  • Clone the project and install dependencies
  • Provide your AWS credentials
  • Install the Asset Tracker Cloud Example into your AWS account
